Create a Repo


1. Create your folder locally
2. Git init (initializes your folder as a git repo)
3. Make a change (add a file etc)
4. git add <file name>
5. git commit -m "add a meaningful commit message"


6. Go to Github and create a new repo
7. Make sure you are in SSH mode
8. Copy the code snippet with the push command (2nd one from the bottom)
9. Paste the command in your terminal
10. Perform the git push command
11. Voila! Look for your success message (100%/100%)
